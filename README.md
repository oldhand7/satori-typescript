A lightweight library to convert HTML JSX to SVG string.

## Quick Start

```jsx
satori(
  <div style={{ color: 'black' }}>hello, world</div>
)
// => '<svg ...><text x="0" y="16" width="45" height="16" fill="black">hello, world</text></svg>'
```
